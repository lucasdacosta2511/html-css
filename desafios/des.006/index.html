Tags HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag usar

  • paragrafos
    Use a tag <p>
  • Titulos
    Use a tag <h1>
  • Imagens
    Use a tag <img>
  • Abreviações
    Use a tag <abbr>
  • Listas numeradas
    Use a tag <ol>
  • Listas com demarcadores
    Use a tag <ul>
  • Para contar o espaçamento
    Use a tag <pre>
